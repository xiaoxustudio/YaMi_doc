Party类（玩家队伍管理器）

处理玩家队伍

内置变量 说明
player 玩家角色对象.
members 玩家队伍成员列表.
version 队伍版本(随着队伍成员的添加和移除发生变化).

方法

方法 说明
reset() 重置队伍角色.
createPlayer() 创建玩家角色.
createMembers() 创建玩家队伍成员.
setPlayer() 设置玩家角色.
addMember() 添加玩家队伍成员.
removeMember() 移除玩家队伍成员.
shareInventory() 共享玩家角色包裹.
saveData() 保存队伍角色数据.
loadData() 加载队伍角色数据.

方法详情

  • reset()

说明：无参数，重置队伍角色

  • createPlayer()

说明：无参数，创建玩家角色

  • createMembers()

说明：无参数，创建玩家队伍成员

  • setPlayer(actor)
参数 类型 说明
actor GlobalActor 玩家角色.

说明：设置玩家角色

  • addMember(actor)
参数 类型 说明
actor GlobalActor 队伍成员.

说明：添加玩家队伍成员

  • removeMember(actor)
参数 类型 说明
actor GlobalActor 队伍成员.

说明：移除玩家队伍成员

  • shareInventory()

说明：共享玩家角色包裹

  • saveData()

说明：保存队伍角色数据

  • loadData(party)
参数 类型 说明
party Object party类型数据.

说明：加载队伍角色数据

Team类（势力队伍管理器）

处理玩家队伍势力

内置变量 说明
key 队伍的键(ID)列表.
relationMaps 队伍关系表(0:敌对, 1:友好).
defaultId 默认队伍键(ID).

方法

方法 说明
buildRelationMaps() 构建角色队伍的关系映射表.
isEnemy() 判断敌对关系.
isFriendly() 判断友好关系.
changeRelation() 改变角色队伍的关系.
saveData() 保存队伍关系数据.
loadData() 加载队伍关系数据.

方法详情

  • buildRelationMaps(keys, code)
参数 类型 说明
key string[] 队伍的ID列表.
code string 队伍的关系代码.

说明：构建角色队伍的关系映射表

  • isEnemy()

说明：判断敌对关系

  • isFriendly()

说明：判断友好关系

  • changeRelation(teamId1, teamId2, relation)
参数 类型 说明
teamId1 string 队伍ID1.
teamId2 string 队伍ID2.
relation number 队伍1和队伍2的关系(0:敌对, 1:友好).

说明：改变角色队伍的关系

  • saveData()

说明：保存队伍关系数据

  • loadData(team)
参数 类型 说明
team object team.

说明：加载队伍关系数据
